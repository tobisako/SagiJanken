詐欺ジャンケン SagiJanken

SAGI JANKEN — HTML5 REMAKE
プレイヤーVSコンピューター

詐欺機能を搭載した、シンプルなジャンケンゲームです。 コンピューターと対戦し、先に5回勝てば勝利。 ——でも、プレイヤーは絶対に勝てない様に作られています。 Visual C++ 2008 製の原作を、プラグイン不要の HTML5 / JavaScript に移植しました。

枠内の「スタート」を押すとその場で遊べます（効果音あり・🔊で消音可） ↗ 全画面で開く

あそびかたHOW TO PLAY

STEP 1
スタート
「じゃ〜んけ〜ん！」の掛け声のあと、手の絵柄が高速で入れ替わります。
STEP 2
グー・チョキ・パー
ボタン（キー1・2・3でも可）で手を出すと、ランダムなかけ声（全10種）とともに勝負。勝敗はAA画像で表示されます。
STEP 3
先に5回勝てば勝利
勝ち数が並んで表示されます。リーチになると展開が変わってきます。負け続けると、ゲームがヒントをくれるかも。

仕掛け（READMEより）THE TRICK

「プレイヤーは絶対に勝てない様に作られています。」——リポジトリREADMEに書かれている仕掛けの全文：
  1. 最初は普通。
  2. プレイヤーがじわじわ勝つ様になっている。
  3. プレイヤーがリーチになると、コンピューターが追い上げてくる。
  4. 両者リーチになると、白熱展開っぽくなる。
  5. 最後はプレイヤーが絶対負ける。

この移植についてFACTS

3モード コンピューターの手を決めるノーマル／詐欺／負けの3モード（ソースの MODE_NOMAL・MODE_SAGI・MODE_MAKE）を切り替えて「じわじわ感」を演出
あいこ5〜9回 両者リーチの最終決戦ではわざと あいこを5〜9回はさんで白熱っぽく（RenewComMode の値そのまま）
5勝勝負に修正 原作の定数は REACH=5・WIN=6 で実質6勝勝負（コメント曰く「勝利数は『6』…5にすればよかった」）。ブラウザ版はコメントどおり 4・5 に修正し、READMEの「先に5回」と一致
かけ声10種 「うりゃっ！」「なん…だと？！」「ごるぁ！！」など、勝負の瞬間のかけ声はランダム10種

効果音はリメイクでの追加です（MOSS-SoundEffect-v2.0 で生成・🔊ボタンでON/OFF）。

原作について（リポジトリREADMEより）
Visual C++ 2008 製（2016年9月公開）。勝敗・リーチなどの画面はAA（アスキーアート）画像、ジャンケンの手は実写画像という構成もそのまま移植しています。「楽しんでください。」
